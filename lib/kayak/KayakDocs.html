ISocket

Represents a socket which supports asynchronous IO operations.

Properties

  • IPEndPoint RemoteEndPoint

    The IP end point of the connected peer.

Methods

  • Task<int> Write(byte[] buffer, int offset, int count)

    Returns an observable which, upon subscription, begins an asynchronous write operation. When the operation completes, the observable yields the number of bytes written and completes.

  • Task WriteFile(string file)

    Returns an observable which, upon subscription, begins copying a file to the socket. When the copy operation completes, the observable completes.

  • Task<int> Read(byte[] buffer, int offset, int count)

    Returns an observable which, upon subscription, begins an asynchronous read operation. When the operation completes, the observable yields the number of bytes read and completes.

KayakServer

KayakServer is a simple server implementation using System.Net.Sockets.Socket. It is an observable which, upon subscription, starts a new thread on which it listens for connections, and yields an ISocket object each time a client connects. Cancelling the subscription causes the listen thread to exit.

ISocket values are yielded on ThreadPool threads. The observables returned by these ISocket values yield and complete on ThreadPool threads as well. Thus, you must take care to synchronize resources shared by concurrent requests.

Constructors

  • KayakServer(IPEndPoint listenEndPoint)

    Constructs a server which binds to port 8080 on all interfaces upon subscription and maintains a default connection backlog count.

  • KayakServer(IPEndPoint listenEndPoint, int backlog)

    Constructs a server which binds to port 8080 on all interfaces upon subscription and maintains a default connection backlog count.
